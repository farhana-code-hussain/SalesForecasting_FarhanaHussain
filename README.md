# 📈 Sales Forecasting Dashboard

## Overview

This project is an end-to-end Machine Learning project for Sales Forecasting and Business Analytics using Python and Streamlit.

## Features

- 📊 Sales Data Analysis
- 📈 Time Series Forecasting
- 🔮 Prophet Forecasting
- 📉 SARIMA Forecasting
- 🤖 XGBoost Forecasting
- 🚨 Anomaly Detection
- 🎯 Product Segmentation using K-Means
- 🌐 Interactive Streamlit Dashboard

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Prophet
- XGBoost
- Streamlit
